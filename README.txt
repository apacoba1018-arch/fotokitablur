===================================================================
✨ TREND FOTO KITA BLUR - OFFLINE AUDIO VERSION ✨
===================================================================

Project ini dimodifikasi khusus untuk menggunakan aset lagu lokal (.mp3)
secara offline tanpa menggunakan sistem streaming YouTube API.

-------------------------------------------------------------------
📂 STRUKTUR FILE PROJECT di dalam ZIP:
-------------------------------------------------------------------
trend-foto-blur-offline/
├── index.html        <- File utama antarmuka web
├── lagu.mp3          <- [WAJIB] Taruh file hasil download/potongan lagu Anda di sini!
├── css/
│   └── style.css     <- Pengaturan animasi & filter canvas
├── js/
│   └── script.js     <- Logika kontrol kamera AI & Trigger Audio HTML5
└── README.txt        <- File panduan ini

-------------------------------------------------------------------

1. Klik "Allow" / "Izinkan" untuk akses webcam perangkat Anda.
2. Tunggu hingga layar preview kamera sebelah kiri muncul.
3. Klik tombol "Mulai Trend" di panel kontrol sebelah kanan.
4. Hitung mundur 3 detik akan berjalan, file `lagu.mp3` akan dimainkan secara lokal, 
   dan acungkan pose jari V ✌️ untuk mengaktifkan filter blur!

Selamat berkreasi! 🚀
